# PowerShell Log Collection Framework

A reusable PowerShell framework for structured IT support log collection.

## Features

- Timestamped report folders
- Standard logging helper
- CSV and JSON exports
- Event log collection by name and time range
- System summary and ticket-ready output

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\PowerShell_Log_Collection_Framework.ps1
```

## Safety

Read-only. It collects local support evidence and does not alter system settings.
